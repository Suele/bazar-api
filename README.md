# Bazar-API

###O que é?
API para venda de produtos.

###Cenário
Em um bazar é possível encontrar diversos tipos de produtos para venda, então será feita a
divisão dos produtos em categorias.

###Recursos da API 
- [X] Mostrar todos os produtos disponíveis paginados.
